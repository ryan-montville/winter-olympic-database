# 2026 Winter Olympic Database
## Datasets
All data comes from [2026 Winter Olympic website](https://www.olympics.com/en/milano-cortina-2026).

The list of athletes was web scaped from the website using python, you can view the code [here](https://github.com/ryan-montville/winter-olympic-database/blob/main/python/athlete_web_scrape.py).
