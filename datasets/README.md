# 2026 Winter Olympic Database
## Datasets
All data comes from [2026 Winter Olympic website](https://www.olympics.com/en/milano-cortina-2026).

You can view the python code used to collect the data and build the csv files [here](https://github.com/ryan-montville/winter-olympic-database/tree/main/python)

Athlete has the headers: athlete_id, athlete_name, gender, date_of_birth, and country_code
coach has the headers: coach_id, coach_name, and sport_name
competes_in has the headers: team_id, event_id, and ranking
country has the headers: country_code, alpha_2, country_name, and continent
event has the headers: event_id, sport_id, event_name, gender_category, date, time, event_type, and venue_id
participates_in has the headers: athlete_id, event_id, and ranking
